Detailed Description

If mode specifies input, configuration can be

  • Analog input
  • Floating input
  • Pull up/down input

If mode specifies output, configuration can be

  • Digital push-pull
  • Digital open drain
  • Alternate function push-pull or analog output
  • Alternate function open drain or analog output
