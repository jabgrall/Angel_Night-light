Le code permet de lancer trois activité selon les besoins :

  • Demo de variation des couleurs.
  • Choix d'une couleur par intéraction avec l'ange. Cette couleur servira de base pour la troisième activité.
  • Enclenchement d'une couleur pulsante activée par une variation du niveau sonore.
