Detailed Description

Structure qui permet de définir les données de la variable.

Deux types de données sont utiles :

  • La fonction caractéristique qui est utile pour le calcul flou ainsi que la fuzzification et la défuzzification des données logiques. La fonction est définit par un tableau qui donne toutes les droites qui définissent la fonction d'appartenance.
  • Le degré d'appartenance qui est utile pour les calculs logiques flous.
