Detailed Description

  • T6: Timer 6 TRGO event
  • T3: Timer 3 TRGO event
  • T8: Timer 8 TRGO event
  • T7: Timer 7 TRGO event
  • T5: Timer 5 TRGO event
  • T15: Timer 15 TRGO event
  • T2: Timer 2 TRGO event
  • T4: Timer 4 TRGO event
  • E9: External line 9
  • SW: Software trigger
Note
: Refer to the timer documentation for details of the TRGO event.
: T3 replaced by T8 and T5 replaced by T15 in some devices.
: this is not valid for the STM32L1 family.
: only used if bit TEN2 is set (DAC channel 1 trigger enabled).
